# Specifications
#### Order of Operations

A randomized binary sequence and players must build a circuit that can output the ordered sequence and then trigger it 

#### Minimum Viable Product

A randomized set of ‘state’ conditions is defined and players must construct a circuit capable of producing the target states and manipulate the objects into said states

#### Process Mapping

A set of input and output nodes are provided for the player. The player must build a circuit that connects them and sends the appropriate signals

#### Blueprint

A partially drawn schematic is provided to players. The player must determine which components would successfully complete the circuit and build it.

#### Bill of Materials

Players are provided a randomized base minimum set of objects that must be used. Players must create a working circuit using these components

#### Sprint

Fixed turn limit; player with the highest score at turn limit wins
